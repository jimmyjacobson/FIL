1. I am not a designer so I apology for any misuse of design terminology here 2. What problem was I trying to solve? My users were putting our embed on their sites, but many of them had responsive layouts. 3. An IFrame renders an HTML document with a set width. 4. Media Queries and other responsive layout techniques can be used and the iframe provides the correct constraints to trigger them. 5. The big problem now becomes, how do you set the height on the iframe? Height 100% on an iframe usually ends up in a 150px tall monstrosity. We want flexible width AND height

Cross Domain IFrame Messaging

  • embedded document detects resize event
  • posts a message to parent window with own height
  • parent window resizes iframe to contain entire embedded document

      • Our needed solution: Cross Domain Iframe Messaging so IFrame can communicate to parent the height of the current document
